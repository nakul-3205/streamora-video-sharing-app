#  Streamora

Streamora is a blazing-fast short video sharing platform, inspired by YouTube Shorts. Built using **Next.js**, **MongoDB**, and **ImageKit**, it allows users to upload videos and explore content shared by others ‚Äî all in a clean, minimal interface.

---

##  Features

-  Upload short videos
-  View others' videos with lightning speed
-  Smart compression via ImageKit
-  MongoDB-powered scalable backend
-  Built with Next.js App Router for modern performance
-  Responsive, clean UI (TailwindCSS + Framer Motion)

> *Note: This is a minimal MVP ‚Äî commenting, liking, and profile features will be added soon.*
